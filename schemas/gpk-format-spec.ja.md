# GalEngine データパック (.gpk) 形式仕様

## 概要

`.gpk`（GalEngine PacKage）形式は、GalEngine ゲームおよびパッチ/DLC のコンパイル済みバイナリ形式です。
ゲーム素材とコンパイル済みスクリプトを単一の配布可能なファイルにパッケージ化します。

## バージョン

現在の形式バージョン：`1.0`

## ファイル構造

```
┌─────────────────────────────────┐
│  ヘッダー (64 バイト)             │
├─────────────────────────────────┤
│  ファイルインデックス (可変長)     │
├─────────────────────────────────┤
│  メタデータブロック               │
│  (JSON, zlib 圧縮)               │
├─────────────────────────────────┤
│  スクリプトブロック               │
│  (コンパイル済みシーンデータ,      │
│  zlib 圧縮)                      │
├─────────────────────────────────┤
│  アセットデータブロック            │
│  (生または暗号化されたアセット)     │
└─────────────────────────────────┘
```

## ヘッダー (64 バイト)

| オフセット | サイズ | フィールド     | 説明                           |
|-----------|------|----------------|--------------------------------|
| 0         | 4    | magic          | `0x4750474B` ("GPKG")         |
| 4         | 2    | version        | 形式バージョン (1)               |
| 6         | 2    | flags          | ビットフラグ（下記参照）          |
| 8         | 8    | index_offset   | ファイルインデックスへのバイトオフセット |
| 16        | 8    | index_size     | ファイルインデックスのバイトサイズ |
| 24        | 8    | meta_offset    | メタデータブロックへのバイトオフセット |
| 32        | 8    | meta_size      | メタデータの圧縮サイズ           |
| 40        | 8    | assets_offset  | アセットデータへのバイトオフセット |
| 48        | 8    | assets_size    | アセットデータの総サイズ         |
| 56        | 8    | reserved       | 将来の使用のために予約           |

### フラグ

| ビット | 名前         | 説明                           |
|-------|-------------|--------------------------------|
| 0     | ENCRYPTED   | アセットが暗号化されている        |
| 1     | PATCH       | これはパッチ/DLCパッケージである  |
| 2     | COMPRESSED  | アーカイブ全体が圧縮されている    |
| 3-15  | RESERVED    | 予約                           |

## ファイルインデックス

ファイルエントリの配列、各エントリ 128 バイト：

| オフセット | サイズ | フィールド   | 説明                           |
|-----------|------|-------------|--------------------------------|
| 0         | 64   | filename    | UTF-8 パス（ヌルパディング）     |
| 64        | 8    | offset      | アセットブロック内のバイトオフセット |
| 72        | 8    | size        | 非圧縮ファイルサイズ             |
| 80        | 8    | compressed  | 圧縮サイズ（生の場合は 0）        |
| 88        | 4    | crc32       | CRC32 チェックサム              |
| 92        | 4    | type        | 0=生, 1=zlib, 2=lzma            |
| 96        | 32   | reserved    | 予約                           |

## メタデータブロック

zlib で圧縮された JSON オブジェクト：

```json
{
  "pack_id": "com.example.mygame",
  "version": "1.0.0",
  "type": "game",
  "engine_version": "0.1.0",
  "title": "My Game",
  "author": "Developer",
  "scenes": ["scene_prologue", "scene_chapter1", ...],
  "languages": ["zh-CN", "en"],
  "dependencies": {
    "base_game": ">=1.0.0"
  },
  "created": "2026-05-31T12:00:00Z"
}
```

## アセットデータブロック

すべてのゲームアセット（画像、音声、フォント）を連結して格納し、ファイルごとにオプションの圧縮が可能です。
ファイルはファイルインデックスにリストされた順序で保存されます。
