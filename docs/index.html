Overview / 概述 / 概要

GalEngine 是一个用 Python 编写的开源视觉小说引擎，支持 JSON 和 Markdown 双格式脚本、内置 Web 图形化编辑器、跨平台运行、.gpk 补丁系统。

GalEngine is an open-source visual novel engine written in Python, featuring dual-format scripts (JSON & Markdown), built-in Web GUI editor, cross-platform support, and .gpk patch system.

GalEngine は Python で書かれたオープンソースのビジュアルノベルエンジンです。JSON と Markdown の両形式スクリプト、内蔵 Web GUI エディタ、クロスプラットフォーム対応、.gpk パッチシステムを特徴とします。

Install / 安装 / インストール

系统要求：Python 3.10+，推荐 3.12+。

System Requirements: Python 3.10+, recommended 3.12+.

システム要件：Python 3.10+、推奨 3.12+。

快速安装

Quick Install

クイックインストール

# Clone the repository
git clone https://github.com/Unk1ndledAC/GalEngine.git
cd GalEngine

# Install dependencies
pip install -r requirements.txt

# Verify
python -m galengine --version

Project Structure / 项目结构 / プロジェクト構造

my_game/
├── settings.json          # Project config (required)
├── assets/
│   ├── backgrounds/     # Background images
│   ├── sprites/         # Character sprites
│   ├── cg/             # CG images
│   ├── audio/
│   ├── fonts/
│   └── ui/
├── scripts/             # Scene scripts
│   ├── prologue.json
│   └── chapter1.md
├── patches/             # Patch files (.gpk)
└── build/               # Build output

settings.json / 项目配置

每个游戏项目必须包含一个 settings.json 配置文件。

Every game project must contain a settings.json config file.

すべてのゲームプロジェクトには settings.json 設定ファイルが必要です。

最小配置示例

Minimal Example

最小構成例

{
  "project": {
    "name": "My Game",
    "version": "1.0.0"
  },
  "assets": {
    "backgrounds": "assets/backgrounds",
    "sprites": "assets/sprites"
  },
  "scenes": {
    "prologue": "scripts/prologue.json"
  }
}

Scene Scripts / 场景脚本

GalEngine 支持 JSON 和 Markdown 两种场景脚本格式。

GalEngine supports both JSON and Markdown scene script formats.

GalEngine は JSON と Markdown の両方のシーンスクリプト形式をサポートします。

Markdown Format / 语法示例

# 序章 / Prologue

==background: bg_room.png==

### Alice

Good morning!

?? What to do?
- [Greet] -> choice_a
- [Ignore] -> choice_b

Compile & Package / 编译打包

# Compile project
python -m galengine.cli compile --project ./my_game

# Package as .exe
python -m galengine.cli package --project ./my_game

# Compile patch
python -m galengine.cli compile-patch --scenes new_chapter

GUI Editor / 图形化编辑器

启动内置 Web 编辑器：

Launch the built-in Web editor:

内蔵 Web エディタを起動：

python -m galengine.editor
# Open http://localhost:8080

FAQ / 常见问题

如何添加人物立绘差分？

How to add sprite variants?

立ち絵のバリエーションを追加するには？

在 assets/sprites/ 目录中，为角色准备多张图片：alice_normal.png、alice_smile.png 等。

In assets/sprites/, prepare multiple images for each character: alice_normal.png, alice_smile.png, etc.

assets/sprites/ ディレクトリで、各キャラクター用に複数の画像を準備します：alice_normal.png、alice_smile.png など。
